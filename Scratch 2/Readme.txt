This is our Scratch 2 extension for Windows, OSX, and Linux.

Go to our website for step by step installation instructions.

http://www.embeditelectronics.com/blog/learn/

If you install as explained above, you can use the Scratch Launcher shortcut to launch Scratch with our extension already running.

To import our Scratch extension into an existing project:

-Plug your PiSoC board into the computer
-Use your file browser to navigate to this folder, and then open "scratch_soc.py"
-Open up Scratch and your existing project
-Shift-click on "File" in Scratch, and then click "Import experimental http extension"
-Navigate to this folder and open "pisoc.s2e"
